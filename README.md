Trigonometry

A C# console application project focused on implementing and practicing trigonometry concepts and mathematical calculations.

This project demonstrates the use of trigonometric functions, formulas, and problem-solving techniques using C#.

---

Features

- Basic Trigonometric Calculations
- Mathematical Formula Implementation
- Console-based User Interaction
- Simple and clean code structure

---

Tech Stack

- C#
- .NET
- Console Application

---

Concepts Covered

- Mathematical Computation
- Console Application Development
- Basic Problem Solving

---

Getting Started

Prerequisites

Make sure you have installed:

- .NET SDK
- Visual Studio / VS Code

---

Clone Repository

git clone https://github.com/MayurKachare8/Trigonometry.git

---

Run Application

Navigate to project directory:

cd Trigonometry

Run the application:

dotnet run
